# Sameer Sahu - Digital Canvas Portfolio

A modern, high-performance glassmorphism portfolio built with React, Vite, and Framer Motion. This project showcases the work of a full-stack developer specializing in AI, ML, and thoughtful software design.

## 🚀 Overview

This portfolio is designed with a sleek "Aurora" aesthetic, featuring smooth animations, responsive layouts, and a clean user experience. It highlights technical skills across the full stack and features selected projects that demonstrate expertise in AI/ML and modern web development.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Radix UI
- **Backend:** Node.js, Express, PostgreSQL, Sequelize, JWT, REST API
- **Tools:** Git, GitHub, Vercel

## ✨ Features

- **Responsive Design:** Optimized for all screen sizes from mobile to desktop.
- **Glassmorphism UI:** Modern, translucent interface elements with smooth backdrop blurs.
- **Aurora Animations:** Dynamic, flowing background gradients powered by Framer Motion.
- **Project Showcases:** Detailed cards for selected works with tech stack tags and links.
- **Skill Groups:** Organized categorization of technical competencies.
- **Timeline:** A chronological view of professional and academic milestones.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sameer-sahu25/FUTURE_FS_01.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
